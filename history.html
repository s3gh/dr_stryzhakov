История создания сайта

29.07.2017 начато создание сайта. Первый сайт создан на adobe muse.

6.08.2017 приобретено доменное имя stryzhakov.com, хостинг на uahosting.com.ua. Планировалось создание сайта на 3 языках, адаптация для планшетов и мобильных устройств.

6.09.2017 сайт запрещен руководством клиники.

10.01.2018 откорректирована последняя версия сайта, сайт перемещен на поддомен test.stryzhakov.com. В начале июня 2020 года поддомен заменен на old.stryzhakov.com, где сайт располагается по сегодняшний день.

10.06.2020 приобретена лицензия Nicepage Personal Edition с целью создания сайта при помощи программы Nicepage.
Изначально планировалось пересоздание сайта на оболочке WordPress либо Joomla. Изучались обе CMS, искались преимущества использования каждой системы. Предпринята попытка создания сайта на WordPress при помощи интегрированной в нее Nicepage. При пересоздании сайта столкнулся с проблемой выравнивания элементов страницы, с еще большей проблемой адаптации сайта для разных разрешений экрана. Дополнительная проблема невозможности подключения скриптов (либо сложности подключения) и, связанных с этим, сложностей точной компоновки элементов.

12.06.20 начато самостоятельное создание шапки сайта на языках html, css, js. Начато изучение языков программирования. Изначально планировалось содать шапку сайта и затем использовать Nicepage и WordPress для дальнейшей разработки сайта. Со временем осознал неизбежность необходимости более глубокого изучения языков программирования...

12.07.20 создана шапка сайта, адаптирована для WordPress.

17.07.20 принято решение отказаться от пересоздания сайта на CMS, начало полное самостоятельное создание сайта на языках программирования html, css, js, php. Создание сайта ведется на украинском языке. Решено русский и английский языки раположить в поддоменах ru.stryzhakov.com и en.stryzhakov.com Позже решено не использовать поддомены, чтобы не нагромождать сайт. Русский и английский языки реализованы через дирректории: stryzhakov.com/ru и stryzhakov.com/eng
